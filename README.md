# US Visa Approval Analysis

## Project Overview

This project analyzes US visa application data to identify factors influencing visa certification and denial. The analysis explores relationships between employee attributes, employer characteristics, and wage levels to understand trends in visa approvals.

## Objectives

* Analyze the distribution of visa approvals and denials
* Identify key factors affecting visa certification
* Explore the impact of education, job experience, wages, and employment type on visa outcomes

## Tools Used

Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Key Insights

* Higher prevailing wages are associated with higher visa certification rates
* Applicants with prior job experience tend to have better approval outcomes
* Full-time positions show higher certification rates than part-time roles

## Conclusion

The analysis highlights important factors influencing visa approval outcomes and provides insights that may help employers better understand trends in visa applications.
